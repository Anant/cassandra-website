Download

The latest stable release of Apache Cassandra is 0.6.1 (released on 2010-04-18).

  • apache-cassandra-0.6.1-bin.tar.gz [PGP] [MD5] [SHA1]
  • apache-cassandra-0.6.1-src.tar.gz [PGP] [MD5] [SHA1]

When downloading from a mirror, please be sure to verify the checksums and OpenPGP compatible signature (see the MD5, SHA1, and PGP links above). The KEYS file contains the public key(s) used for signing releases.

New users to Cassandra should be sure to check out the "Getting Started" guide.

Source Control

The Cassandra source code resides in the Apache SVN repository. To check out a copy:

svn checkout https://svn.apache.org/repos/asf/cassandra/trunk cassandra

There is also a read-only git repository if that is more your style.

git clone git://git.apache.org/cassandra.git

Potential contributors should look over the wiki, in particular HowToContribute.
