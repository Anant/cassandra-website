Cassandra Server

Cassandra releases include the core server, the nodetool administration command-line interface, and a development shell (cqlsh and the old cassandra-cli).

The latest stable release of Apache Cassandra is 2.0.4 (released on 2013-12-30). If you're just starting out, download this one.

Apache provides binary tarballs and Debian packages:
  • apache-cassandra-2.0.4-bin.tar.gz [PGP] [MD5] [SHA1]
  • Debian installation instructions

Third Party Distributions

  • DataStax Community is available in Linux rpm, deb, and tar packages, a Windows MSI installer, and a Mac OS X binary.

CQL

The current major version of the Cassandra Query Language is CQL version 3. The CQL3 documentation for the current version of Cassandra (2.0) is available here. The documentation for CQL3 in Cassandra 1.2 is here.

New users to Cassandra should be sure to check out the Getting Started guide.

Previous and Archived Cassandra Server Releases

Previous stable branches of Cassandra continue to see periodic maintenance for some time after a new major release is made. The lastest release on the 1.2 branch is 1.2.14 (released on 2013-02-03).

  • apache-cassandra-1.2.14-bin.tar.gz [PGP] [MD5] [SHA1]

The lastest release on the 1.1 branch is 1.1.12 (released on 2013-05-27).

  • apache-cassandra-1.1.12-bin.tar.gz [PGP] [MD5] [SHA1]

Older (unsupported) versions of Cassandra are archived here.

Source

The Cassandra source code is available to download:

  • apache-cassandra-2.0.4-src.tar.gz [PGP] [MD5] [SHA1]
  • apache-cassandra-1.2.14-src.tar.gz [PGP] [MD5] [SHA1]
  • apache-cassandra-1.1.12-src.tar.gz [PGP] [MD5] [SHA1]
Development is done in the Apache Git repository. To check out a copy:

git clone http://git-wip-us.apache.org/repos/asf/cassandra.git

Bleeding edge development snapshots of Cassandra are available from Jenkins continuous integration.

Potential contributors should look over the wiki, in particular HowToContribute.
