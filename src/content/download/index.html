Tick-Tock Cassandra Server Releases

Cassandra is moving to a monthly release process called Tick-Tock. Even-numbered releases (e.g. 3.2) contain new features; odd-numbered ones (e.g. 3.3) contain bug fixes only. If a critical bug is found, a patch will be released against the most recent bug fix release. Read more about tick-tock here.
